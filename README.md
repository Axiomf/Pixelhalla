# pixelhalla

1.  **نصب dependencyها:**

    کتابخانه‌های مورد نیاز پروژه را با استفاده از فایل `requirements.txt` نصب کنید:
    ```bash
    pip install -r requirements.txt
    ```

2.  **اجرای بازی**

    بازی را با اجرای اسکریپت `run_game.py` شروع کنید:
    ```bash
    python run_game.py
    ```
